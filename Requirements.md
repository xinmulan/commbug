```
   autotools-dev, 

   mono-devel,  

   libglade2.0-cil-dev,   

   libmono-addins-cil-dev (>= 0.3.1),

   libmono-addins-gui-cil-dev (>= 0.3.1),

   libglib2.0-cil-dev,

   libgtk2.0-cil-dev (>= 2.12), 

   gconf2,

   libglib2.0-dev,

   libgtk2.0-dev (>= 2.8)
```

```
   mono

   libmono-i18n2.0-cil

   libgdiplus
```

```
   .NET Framework (>= 2.0)

   gtk-sharp (>= 2.0)
```

```
   X11
```
